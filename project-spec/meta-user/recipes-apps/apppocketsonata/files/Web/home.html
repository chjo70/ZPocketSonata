정비 소프트웨어

이 웹 브라이저의 CGI(Common Gateway Interface) 를 통하여 ZYNQ 보드에 실행중인 운용 소프트웨어의 쓰레드 통신하여 데이터를 확인하거나 로컬에 저장된 각종 파일 시스템을 억세스 하여 다양한 기능을 제공 합니다.

또한, 운용 소프트웨어의 롬화 파일을 업로드할 수 있으며 운용 시스템의 세부 설정 값을 확인할 수 있습니다.

  • 시스템 변수: 하드웨어 및 소프트웨어 설정 값을 설정
  • 자체 점검: 초기/자체/운용자 자체 점검 결과를 수행
  • 다운로드: 저장된 데이터를 다운로드
  • 업로드: 시스템 변수 설정 파일, 운용 프로그램을 업로드
  • 개발 지원: 분석 결과 확인
  • 프로젝트 개발: 최종 소스 이력을 확인

 

소형 함정 전자전 장비의 소스는 여기를 클릭해서 클래수 및 함수를 확인할 수 있습니다.

수행 에러나 질문 사항이 있으시면  로 보내주시면 고맙겠습니다.

 

본 웹 페이지는 Internet Explorer , 크롬, 모질라 등 다양한 웹 브라우저에서 사용할 수 있습니다.
